# BEW-1-2_bluedit
Bluedit is a Reddit clone using some cool backend features including:
- Node.js
- Express.js
- Handlebars
- Bootstrap
- Mongo DB
	- mLab
	- Mongoose
- Bcrypt

## Why?
Bluedit was created for Make School's backend 1.2 course.
